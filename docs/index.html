Get started in a minute

Mixtape is distributed directly, so the first launch takes one extra click.

Download the latest .dmg from the releases page and drag Mixtape to your Applications folder.

Right-click the app → Open the first time (a one-time step for this un-notarised build), then click Open again to confirm.

Updates are automatic. Mixtape checks for new versions and updates itself from then on, with no re-downloading.
